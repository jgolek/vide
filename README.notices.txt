-------------
Vide
-------------
Web application generation tool

1. What's the use case of this app or library?

With vide you can simply and fast build an web application. 
It's contains
 * reuse of front end Widgets
 * server client communication about REST
 * database storage integration
 * user authentification
 * multi and single page apps
 * on page widgets communction
